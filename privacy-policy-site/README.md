# LottoPilot Privacy Policy - GitHub Pages

## 1. 修改联系邮箱

打开 `index.html`，将所有 `YOUR_EMAIL@example.com` 替换为你的真实邮箱。

## 2. 部署到 GitHub Pages

1. 在 GitHub 新建仓库（例如 `lottopilot-privacy`）
2. 将本文件夹中的 `index.html` 和 `auth/callback/` 目录上传到仓库
3. 进入仓库 **Settings** → **Pages**
4. **Source** 选择 **Deploy from a branch**
5. **Branch** 选择 `main`（或 `master`），路径选 `/ (root)`
6. 保存后等待 1–2 分钟

## 3. 获取链接

页面地址格式：`https://你的用户名.github.io/lottopilot-privacy/`

例如：`https://oldhen666.github.io/lottopilot-privacy/`

将该链接填入 Play Console → 政策 → 应用内容 → 隐私政策。
